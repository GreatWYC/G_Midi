git pull --rebase origin main# G_Midi

- A Midi parsing package written in Java.

### Getting Started
  ```Java
  import com.greatwyc.G_Midi;
  ```

### How To Use

#### G_Midi(File f)
* 将一个midi文件实例化一个G_Midi类
  ```Java
  G_Midi(File f)
  ```

#### List<List> getMessageList()
* 用于获取当前midi消息列表
  ```Java
  List<List> getMessageList()
  ```

#### double getSpeed()
* 用于获取当前midi速度
  ```Java
  double getSpeed()
  ```
